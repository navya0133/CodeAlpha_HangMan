# 🎮 Hangman Game

## 📌 Project Description
Hangman Game is a simple text-based Python game where the player guesses a hidden word one letter at a time. The player has 6 incorrect attempts to guess the correct word.

## 🚀 Features
- Random word selection
- Letter guessing system
- Input validation
- Win/Lose conditions
- Simple console interface

## 🛠️ Technologies Used
- Python
- Random Module

## 📚 Concepts Used
- random.choice()
- while loop
- if-else statements
- strings
- lists
- user input handling

## ▶️ How to Run
1. Install Python
2. Save the file as `hangman.py`
3. Run the command:

```bash
python hangman.py
